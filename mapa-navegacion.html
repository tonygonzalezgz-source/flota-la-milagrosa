🔄 Flujos de Navegación Principales

Flujo 1 — Analista registrando pasajeros del día:

Login → Dashboard Analista → Listado Vehículos → Seleccionar Bus → Form. BEA → Guardar

Flujo 2 — Mantenimiento registra cambio de aceite:

Login → Dashboard Mant. → Listado Vehículos → Ficha Técnica → Form. Cambio Aceite → Guardar

Flujo 3 — Mantenimiento reporta novedad mecánica:

Dashboard Mant. → Ficha del Bus → Form. Novedad → Asigna severidad → Guardar

Flujo 4 — Propietario revisa ingresos de sus buses:

Login → Dashboard Propietario → Mis Vehículos → Seleccionar Bus → Detalle Ingresos → Comparar / Exportar

Flujo 5 — Propietario revisa estado mecánico:

Dashboard Propietario → Estado Mecánico → Detalle del Bus → Historial de Mant.
